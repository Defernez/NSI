Introduction à la Création de Pages Web

Notions de Base

Les pages web sont créées en utilisant un langage appelé HTML (Hypertext Markup Language). HTML permet de structurer le contenu d'une page en utilisant des balises.

En savoir plus sur HTML

Création d'une Page Simple

Pour créer une page web simple, il vous suffit d'un éditeur de texte et de quelques balises HTML de base. Un exemple de balise est <p> pour les paragraphes.
